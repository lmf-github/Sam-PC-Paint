SAM Coupé SCREEN$ Editor

A hardware-accurate pixel editor for the SAM Coupé home computer.

Supported modes: Mode 3 (512×192, 4 col) and Mode 4 (256×192, 16 col).

Keyboard shortcuts:
P=Pencil, B=Brush, L=Line, R=Rect, E=Ellipse, F=Fill,
X=Eraser, I=Eyedropper, S=Select, T=Text
Ctrl+Z=Undo, Ctrl+Y/Ctrl+Shift+Z=Redo
[ / ] = Brush size, G=Toggle Grid
Scroll = Zoom

Left-click palette = set FG · Right-click palette = set BG
Left-click CLUT entry = select FG · Right-click = select BG
Shift-click CLUT entry = reassign slot from full palette
