Web screen editor to assist / reduce workflow in generating Sam Coupe SCREEN$ files. 
Allows you to open, edit and save MODE 1,2,3 or MODE 4 SCREEN$.

    -Can open, load and save directly to Sam disk.
    -Allows additon of Line interupts.
    -Allows conversion of existing BMP, allowing for inclusion of line interupts to increase colour count.

Very much a Claude generated product, verified against the SAMDOS source, SimCoupe emulator source, and scrimage/palette.py
Seeing as it is very much AI assisted, the code may be messy, poorly constructed or hard to maintain, but ewven so, it will be a lot better than anything I could produce on my own!
Technical details provided in this repository are a summary, or are assumed technical details, and should not be relied upon. Always check primary sources.

Happy to take constructive comments on features that may be useful. Its something that I've found saves me time and avoids the need for multiple tools and steps in getting graphics onto the Sam.
